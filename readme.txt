This plugin will take a title and generate a Bob Ross quote

== Description ==

This is a plugin that is designed to genearte a 3-paragrah long Bob Ross quote taken from Bob Ross Lorem Ipsum (https://www.bobrosslipsum.com/). 

This plugin takes an optional title argument. Leave the title argument out to get "Bob Ross Quote" in a header tag above the Bob Ross quote.

== Example ==

<code>[H_BobRoss title='Bob Ross Speaks']</code>
